# `sdk` example

## Usage

### Prerequisites

- [**componentize-go**](https://github.com/bytecodealliance/componentize-go) - Latest version
- [**go**](https://go.dev/dl/) - v1.25.9
- [**wasmtime**](https://github.com/bytecodealliance/wasmtime)  - v44.0.1

### Run

```sh
make run

# Invoke the application
curl localhost:6767
```
